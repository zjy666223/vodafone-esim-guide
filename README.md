# 荷兰 Vodafone eSIM 整理站

这是一个静态网页项目，用中文重新整理公开页面中关于荷兰 Vodafone 预付费 eSIM 的关键信息，并做了新的排版与视觉设计。

## 特点
- 适合直接部署到 GitHub Pages
- 单文件静态页面，维护简单
- 内容做了结构重组与改写，不是原文镜像
- 补充了保号成本小计算器

## 本地预览
直接双击 `index.html` 即可，或使用任意静态服务器预览。

## 说明
- 核心信息参考自公开页面：<https://potoh.com/posts/2026-1-5/wdfesim/>
- 实际资费与规则请以官方页面和运营商政策为准
